Quick Start & Datasets

To instantly verify the code execution, the completely preprocessed ML-1M dataset is included. You can directly run the model using:
python main.py

Due to GitHub's file size limits, the other three larger datasets (Toys, Beauty, Yelp2020) are not included. To evaluate on these datasets, please download the raw data from their official sources and strictly follow the preprocessing protocol detailed in Section 5.1 of our manuscript.
